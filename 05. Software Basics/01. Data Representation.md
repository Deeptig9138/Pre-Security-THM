# 💾 Data Representation 

## 📘 Overview

The **Data Representation** module explains how computers store and interpret:

- Colors  
- Numbers  
- Binary values  
- Hexadecimal values  

Although humans naturally use the **decimal system (0–9)**, computers only understand two states:

- `0`
- `1`

> 💡 **Core Idea:**  
> Everything inside a computer — colors, numbers, text, and files — is ultimately represented using binary digits.

## 🧩 Task 1: Introduction

Humans use the **decimal (base-10)** system in daily life.

Computers use the **binary (base-2)** system because electronic devices operate using two states:

- ON / OFF  
- High / Low voltage  
- True / False

### 🧠 Key Idea

A computer represents information using:

- **Bits** → Single binary digit (`0` or `1`)
- **Bytes** → Group of 8 bits

## 🎨 Task 2: Representing Colors

### 🌈 RGB Color System

Computers create colors using combinations of:

- Red  
- Green  
- Blue  

This is called the **RGB model**.

### 🎨 First 8 Colors

If each color can only be:

- ON (`1`)
- OFF (`0`)

Then:

- 2 × 2 × 2 = 8 possible colors

### 🧾 Color Examples

| Binary | Meaning | Color |
|---|---|---|
| `000` | All OFF | Black |
| `001` | Blue ON | Blue |
| `010` | Green ON | Green |
| `100` | Red ON | Red |
| `011` | Green + Blue | Cyan |
| `101` | Red + Blue | Magenta |
| `110` | Red + Green | Yellow |
| `111` | All ON | White |

### 🧠 Bit and Byte

#### Bit

- Smallest unit of data
- Can be `0` or `1`

#### Byte

- 8 bits
- Can represent 256 values

### 🌈 16 Million Colors

Modern systems use:

- 8 bits for Red
- 8 bits for Green
- 8 bits for Blue

Total:

- 24 bits (3 bytes)

This allows:

- 256 × 256 × 256 = 16,777,216 colors

### 🔢 Hexadecimal Representation

Hexadecimal simplifies binary representation.

#### Example

## 🔢 Task 3: Numbers – From Decimal to Hexadecimal

### 🧠 Decimal System (Base-10)

Humans use:

```text
0–9
```

Example:

```text
213 = 2×10² + 1×10¹ + 3×10⁰
```

### 💻 Binary System (Base-2)

Computers use:

```text
0 and 1
```

Each position represents a power of 2.

### 🔍 Binary Example

Binary:

```text
1001
```

Calculation:

```text
1×2³ + 0×2² + 0×2¹ + 1×2⁰
= 8 + 0 + 0 + 1
= 9
```

### 🔢 More Binary Examples

| Binary | Decimal |
|---|---|
| 0000 | 0 |
| 0001 | 1 |
| 0010 | 2 |
| 0011 | 3 |
| 1100 | 12 |
| 1101 | 13 |
| 1110 | 14 |
| 1111 | 15 |

### 🔠 Hexadecimal Numbers

Hexadecimal uses:

```text
0–9 and A–F
```

#### Hex Values

| Hex | Decimal |
|---|---|
| A | 10 |
| B | 11 |
| C | 12 |
| D | 13 |
| E | 14 |
| F | 15 |

### 🧠 Why Hexadecimal?

Hexadecimal is easier for humans to read than long binary numbers.

#### Example

Binary:

```text
11111111
```

Hexadecimal:

```text
FF
```

### 🧮 Hexadecimal to Decimal Example

```text
9BDF
```

Calculation:

```text
9×16³ + 11×16² + 13×16¹ + 15×16⁰
= 39903
```

### 🧾 Octal System (Optional)

## 🔑 Key Terminology

### Decimal (Base-10)

- Human number system
- Uses digits `0–9`

### Binary (Base-2)

- Computer number system
- Uses `0` and `1`

### Hexadecimal (Base-16)

- Uses `0–9` and `A–F`
- Every hex digit = 4 bits

### Octal (Base-8)

- Uses digits `0–7`
- Every octal digit = 3 bits

### Bit

- Single binary digit
- `0` or `1`

### Byte

- 8 bits
- Also called an **octet**

### Hex Color

- RGB color represented in hexadecimal
- Uses 3 bytes:
  - Red
  - Green
  - Blue
