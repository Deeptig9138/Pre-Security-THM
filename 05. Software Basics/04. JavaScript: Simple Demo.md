## 🧩 Task 2: Variables

### 🧠 What is a Variable?

A variable stores data that can change while a program runs.

In our game we need:

- `tries` → number of attempts
- `guess` → current user guess

### 📦 Declaring Variables

JavaScript uses the keyword `let`.

```javascript
let tries = 0;
let guess = 0;
```

### 🧠 Why Start at Zero?

- User has made zero attempts
- Guess starts with an impossible value
- Prevents accidental success before input

### 🔒 Constants

#### 🧠 What is a Constant?

A constant stores a value that should never change.

JavaScript uses the keyword `const`.

#### 🎲 Creating a Secret Number

```javascript
const secret = Math.floor(Math.random() * (20)) + 1;
```

### 🔍 How It Works

#### Math.random()

Generates a decimal number:

```javascript
0.372...
```

#### Multiply by 20

```javascript
7.44...
```

#### Math.floor()

Removes the decimal portion:

```javascript
7
```

#### Add 1

Moves range from:

```text
0–19
```

to

```text
1–20
```

### 🎯 Result

The computer randomly chooses:

```text
1, 2, 3, ... 20
```

### 🖥️ Displaying Output

To display text on the screen, JavaScript uses:

```javascript
console.log("I'm thinking of a number between 1 and 20");
```

**Q)** What word declares a constant?  
**A)** `const`

**Q)** Method used to display text?  
**A)** `console.log()`

## 🧩 Task 3: Prompting the User for Input

### 🧠 Getting User Input

Node.js requires a special library to receive keyboard input.

The program asks:

```javascript
const text = await rl.question("Take a guess: ");
```

### 🔄 Converting Text to a Number

User input arrives as text.

Convert it to an integer using:

```javascript
guess = parseInt(text, 10);
```

### 🧠 Why Use parseInt()?

Input:

```text
"15"
```

becomes:

```text
15
```

which allows numerical comparisons.

### 📚 Supporting Libraries

JavaScript imports modules to handle command-line interaction.

```javascript
import * as readline from "node:readline/promises";
import { stdin as input, stdout as output } from "node:process";
```

### 🎤 Creating the Interface

```javascript
const rl = readline.createInterface({ input, output });
```

Think of this as creating a communication channel between:

- Keyboard (input)
- Screen (output)

### 🔒 Cleaning Up

After finishing:

```javascript
rl.close();
```

This closes the interface properly.

4. Display game message
5. Ask user for a guess
6. Convert text to number

## 🧩 Task 4: Conditional Statements

### 🧠 Why Conditionals?

The program needs to respond differently based on the user's guess.

### 🔀 Decision Logic

#### Case 1

Guess outside range:

```text
Less than 1
or
Greater than 20
```

➡️ Out of range

#### Case 2

Guess smaller than secret

➡️ Too low

#### Case 3

Guess larger than secret

➡️ Too high

#### Case 4

Guess equals secret

➡️ Success

### 💻 JavaScript Implementation

```javascript
if (guess < 1 || guess > 20) {
    console.log("That number is out of range. Try again.");
} else if (guess < secret) {
    console.log("Too low, try again.");
} else if (guess > secret) {
    console.log("Too high, try again.");
} else {
    console.log("You got it in", tries, "tries!");
}
```

#### Greater Than

## 🧩 Task 5: Iterations

### 🧠 The Problem

Current version only allows one guess.

That does not make for a fun game.

### 🔁 Solution: While Loop

A loop repeats code while a condition remains true.

### 💻 While Loop Syntax

```javascript
while (guess !== secret) {
    // repeat code
}
```

### 🧠 What Does !== Mean?

```javascript
!== 
```

means:

```text
Not Equal To
```

### 🔄 Loop Logic

As long as the guess is wrong:

1. Ask for input
2. Convert input to number
3. Increment tries
4. Compare values
5. Display hint
6. Repeat


### 💻 Full Loop

```javascript
while (guess !== secret) {
    const text = await rl.question("Take a guess: ");
    guess = parseInt(text, 10);

    tries = tries + 1;

    if (guess < 1 || guess > 20) {
        console.log("That number is out of range. Try again.");
    } else if (guess < secret) {
        console.log("Too low, try again.");
    } else if (guess > secret) {
        console.log("Too high, try again.");
    } else {
        console.log("You got it in", tries, "tries!");
    }
}
```
