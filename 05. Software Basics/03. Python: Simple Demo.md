### 🎮 Example Project: Guess the Number Game

The goal is to build a simple game:

1. Computer picks a secret number (1–20)  
2. User guesses repeatedly  
3. Computer gives hints  
   - Too high  
   - Too low  
4. Game ends when guessed correctly

```

Take a guess: 10
Too high, try again.

```

## 🧩 Task 2: Variables

### 🧠 What are Variables?

Variables are **containers for storing data**.

In our game, we use:

- `secret` → the hidden number  
- `guess` → user input  
- `tries` → number of attempts  

### 🎲 Generating a Secret Number

Python provides a random library:

```python
import random
secret = random.randint(1, 20)
``` 

### 📦 Initial Variables

```python
tries = 0
guess = 0
``` 

We start `guess` at 0 so it cannot accidentally match the secret number.

### 🧾 Display Message

```python
print("I'm thinking of a number between 1 and 20")
``` 

### ⌨️ Getting User Input

```python
text = input("Take a guess: ")
guess = int(text)
``` 

### ➕ Counting Attempts

```python
tries = tries + 1
``` 

### 🧠 Answer Check

**Q)** Function to display text?  
**A)** `print()`

**Q)** Function to convert input to integer?  
**A)** `int()`

## 🧩 Task 3: Conditional Statements

### 🧠 What are Conditionals?

Conditionals let the program **make decisions**.

### 🔀 Logic Flow

- If guess is invalid → out of range  
- Else if guess is too low → hint  
- Else if guess is too high → hint  
- Else → correct  

### 🧾 Python Implementation

```python
if guess < 1 or guess > 20:
    print("That number is out of range. Try again.")
elif guess < secret:
    print("Too low, try again.")
elif guess > secret:
    print("Too high, try again.")
else:
    print("You got it in", tries, "tries!")
``` 

### 🧠 Key Concept

- `if` → first condition  
- `elif` → else if  
- `else` → fallback  

### 🧠 Answer Check

## 🔁 Task 4: Iterations (Loops)

### 🧠 Why Loops?

Loops allow repetition until a condition is met.

### 🎯 Game Requirement

User must keep guessing until correct number is found.

### 🔁 While Loop

```python
while guess != secret:
``` 

This means:

> Keep running while the guess is NOT equal to the secret.

### 🎮 Full Loop Logic

```python
while guess != secret:
    text = input("Take a guess: ")
    guess = int(text)

    tries = tries + 1

    if guess < 1 or guess > 20:
        print("That number is out of range. Try again.")
    elif guess < secret:
        print("Too low, try again.")
    elif guess > secret:
        print("Too high, try again.")
    else:
        print("You got it in", tries, "tries!")
``` 

### 🧠 Loop Behavior

- Keeps repeating until correct guess  
- Stops when condition becomes false  

### 🧠 Answer Check

**Q)** Type of loop used?  
**A)** `while`

**Q)** Output if guessed correctly in 3 tries?
