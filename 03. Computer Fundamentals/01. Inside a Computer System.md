# 🖥️ Inside a Computer System 

The **Inside a Computer System** module introduces the fundamental building blocks of a computer and how they work together.

Before diving into cybersecurity, it's essential to understand **what you're protecting**. Think of a computer like a castle — you must know its structure, components, and flow before defending it.

This module provides a **beginner-friendly foundation** to help you understand how a computer system operates internally.

---

## 🎯 Learning Objectives

After completing this module, you will be able to:

- Identify core components of a computer system  
- Understand the role of each component  
- Explain how these components interact  
- Describe the computer boot process

## ⚙️ Task 2: Inside a Computer System

Every computer system consists of essential components working together.

![PC components](https://tryhackme-images.s3.amazonaws.com/user-uploads/66c44fd9733427ea1181ad58/room-content/66c44fd9733427ea1181ad58-1770828504619.svg)

### 🧱 Core Components

- **CPU (Central Processing Unit)** – The brain of the computer  
- **RAM (Memory)** – Temporary working space  
- **Storage (HDD/SSD)** – Permanent data storage  
- **Motherboard** – Connects all components  
- **Power Supply (PSU)** – Provides power  
- **Input Devices** – Keyboard, mouse  
- **Output Devices** – Monitor, speakers  

### 🧠 Human Body Analogy

| Computer Component | Human Equivalent |
|------------------|-----------------|
| CPU              | Brain           |
| RAM              | Short-term memory |
| Storage          | Long-term memory |
| PSU              | Heart           |
| Motherboard      | Nervous system  |

> 🧠 **Security Insight:**  
> Each component can be a potential attack surface if misused or compromised.

## ⚡ Task 3: What Happens When You Press the Start Button?

This task explains the **boot process** — how a computer starts.

![Boot Sequence Diagram](https://tryhackme-images.s3.amazonaws.com/user-uploads/66c44fd9733427ea1181ad58/room-content/66c44fd9733427ea1181ad58-1770828785615.svg)

### 🔄 Boot Sequence

**Step 1: Power Button**
- Sends signal to PSU  
- Power flows through the system  

**Step 2: Firmware Starts (UEFI/BIOS)**
- Initializes hardware components  
- Prepares system startup  

> ℹ️ BIOS is older; UEFI is modern replacement

**Step 3: Power-On Self Test (POST)**
- Checks hardware:
  - CPU  
  - RAM  
  - Storage  
- Detects failures early  

**Step 4: Select Boot Device**
- Chooses device to load OS from:
  - Hard Drive  
  - SSD  
  - USB  

**Step 5: Bootloader Execution**
- Loads Operating System into RAM  
- Transfers control to OS  

> 🧠 **Security Insight:**  
> The boot process is a common target for advanced attacks.
