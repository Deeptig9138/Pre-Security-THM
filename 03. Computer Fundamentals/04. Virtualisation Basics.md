# 🧠 Virtualisation Basics – Learning Path

## 📘 Overview

The **Virtualisation Basics** module explains how modern systems efficiently use hardware through **virtualization**.

Instead of running one application per physical server, virtualization allows multiple systems to share the same hardware — reducing cost, improving efficiency, and enabling scalability.

> 💡 **Core Idea:**  
> One physical machine can behave like many independent computers.

# 🧱 Task 3: Virtualization Components

### 🧠 Hypervisor (The Manager)

A **hypervisor**:

- Creates and manages VMs  
- Allocates CPU, RAM, storage  
- Ensures isolation between systems  

### 🔄 Types of Hypervisors

| Type | Description | Use Case |
|------|------------|---------|
| Type 1 | Runs on hardware | Servers, data centers |
| Type 2 | Runs on OS | Learning, testing |

### 💻 Virtual Machines (VMs)

A **Virtual Machine** is:

- A complete virtual computer  
- Has its own:
  - CPU  
  - RAM  
  - Storage  
  - OS  

### 🔐 Key Feature

- Fully isolated from other VMs  

### 📦 Containers (Lightweight Environments)

A **container**:

- Runs a single application  
- Shares the host OS kernel  
- Starts quickly  
- Uses fewer resources  

### ⚖️ VM vs Container

| Feature | VM | Container |
|--------|----|----------|
| OS | Full OS | Shared OS |
| Size | Large | Lightweight |
| Speed | Slower | Fast |
| Isolation | Strong | Moderate |

### 🐳 Docker

- Popular platform for containers  
- Simplifies deployment  
- Ensures consistency across systems  

### ✅ Answers

**Q)** Best hypervisor for personal lab?  
**A)** `type 2`

**Q)** Best for multiple apps in one VM?  
**A)** `containers`

## 🧠 Task 5: Conclusion

Virtualization is a **core technology** behind modern IT systems.

### 🔑 Key Terminology

- **Virtualization** → One system acting as many  
- **Hypervisor** → Manages virtual systems  
- **VM** → Full virtual computer  
- **Container** → Lightweight app environment  
- **Container Image** → Template for containers  
- **Ports** → Communication endpoints  

### 🚀 Benefits of Virtualization

- Cost savings  
- Efficient resource usage  
- Faster deployment  
- Scalability  
- Flexibility  
- Safe testing environments  
- Centralized management
