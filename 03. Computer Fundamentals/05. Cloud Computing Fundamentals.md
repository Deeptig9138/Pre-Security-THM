# ☁️ Cloud Computing Fundamentals 

## 📘 Overview

The **Cloud Computing Fundamentals** module explains how modern applications are built, deployed, and scaled using the **cloud**.

Running an application on a single computer limits:

- Performance  
- Availability  
- Global access  
- Scalability  

Cloud computing solves these problems by allowing you to use **shared computing resources over the internet**.

> 💡 **Core Idea:**  
> The cloud lets you run applications anywhere, anytime, at scale.

## ☁️ Task 2: Cloud Computing Overview

### 📈 Evolution of Computing

- Physical Servers  
- Virtualization  
- Containers  
- Cloud Computing  

![Cloud evolution](https://tryhackme-images.s3.amazonaws.com/user-uploads/68dac5d6d4d4f23175b3296f/room-content/68dac5d6d4d4f23175b3296f-1768314591776.svg)

### 🌟 Cloud Benefits

- **Scalability** → Adjust resources anytime  
- **On-demand** → Create resources instantly  
- **Pay-as-you-go** → No upfront costs  
- **Security** → Provider-managed protection  
- **High availability** → Minimal downtime  
- **Global access** → Reach users worldwide  

### 🌍 Cloud Deployment Types

| Type | Description | Use Case |
|------|------------|---------|
| Public Cloud | Shared infrastructure | Startups, web apps |
| Private Cloud | Dedicated environment | Banks, government |
| Hybrid Cloud | Mix of both | E-commerce |

### 🧱 Cloud Service Models

| Model | Description | Responsibility |
|------|------------|---------------|
| IaaS | Rent infrastructure | You manage OS + apps |
| PaaS | Managed platform | Focus on app development |
| SaaS | Ready-to-use software | Provider manages everything |

### 🏢 Analogy

- IaaS → Renting an empty apartment  
- PaaS → Furnished apartment  
- SaaS → Hotel service  

### 🏢 Major Cloud Providers

- Amazon Web Services (AWS)  
- Microsoft Azure  
- Google Cloud Platform (GCP)  
- Alibaba Cloud  
- IBM Cloud  
- Oracle Cloud  

### 🌍 Real-World Usage

- **Netflix** → Streaming at global scale  
- **Spotify** → Music delivery  
- **Instagram** → Media storage & delivery  
- **E-commerce** → Handle traffic spikes  

### ✅ Answers

**Q)** What handles sudden traffic increases?  
**A)** `Scalability`

**Q)** Most common cloud type?  
**A)** `public cloud`

**Q)** Best model for focusing only on development?  
**A)** `PaaS`

## 🚀 Task 3: Deploying a Cloud Instance

Now you simulate deploying infrastructure in the cloud.

### 🧠 Key Concepts

### EC2 (Virtual Server)

- Cloud-based computer  
- Has CPU, RAM, storage  

### Instance Types

- Small → Low cost, low power  
- Large → High cost, high power  

### 🌍 Region Selection

- Choose geographic location  
- Impacts latency and performance  

### 🖥️ Created Instances

### Application Interface

- Name: `application-interface`  
- Type: `t3.micro`  
- Status: Running  

### Study Machines

- `study-machine-1` → `m5.large`  
- `study-machine-2` → `m5.large`  

### 💰 Cost Optimization

- Stop unused instances  
- Reduce billing instantly  

> 🧠 **Key Insight:**  
> Cloud lets you control cost dynamically.

### ✅ Answers

**Q)** Total cost after stopping study machines?  
**A)** `30`

**Q)** Cost of `m5.large` per month?  
**A)** `70`

**Q)** Cost if only new instances run?  
**A)** `150`

**Q)** Cost after adding `t3a.small`?  
**A)** `188`
