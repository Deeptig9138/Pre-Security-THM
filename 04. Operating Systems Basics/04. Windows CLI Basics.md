The **Windows CLI Basics** module introduces you to the **Windows Command Prompt (CMD)** and teaches you how to interact with Windows systems using text-based commands instead of graphical menus.

- Navigate files quickly  
- Search for hidden information  
- Read files directly  
- Gather system details  
- Troubleshoot efficiently

### 🖥️ What Is the Windows Command Line?

The **Command Prompt (CMD)** is a text-based interface used to interact with Windows.

Instead of clicking:

- You type commands  
- Windows executes them directly  

### ⚡ Why Use the CLI?

- Faster than GUI navigation  
- Greater control  
- Essential for cyber security tasks  
- Many security tools require terminal usage

## 📂 Task 2: Navigating Files and Finding Your First File

### 🧠 Terminal Basics

The terminal allows direct interaction with the Windows operating system.

Cyber security professionals use it daily for:

- Investigation  
- Troubleshooting  
- Automation  
- File analysis  

### 📍 Step 1: Check Current Directory

#### Command

```
cd
```

#### Purpose
Displays your current working directory.

### 📁 Step 2: List Files and Folders

#### Command

```
dir
```

#### Purpose
Lists files and folders in the current directory.

### 👀 Step 3: Show Hidden Files

#### Command
```
dir /a
```

#### Purpose
Displays all files, including hidden items.

> 💡 Hidden files are not necessarily secret — Windows simply hides them by default.

### 🚶 Step 4: Move Between Directories

#### Change Directory
```
cd Documents
```

#### Move Back One Level
```
cd ..
```

### 🔍 Step 5: Search for a File

#### Command
```
dir /s task_brief.txt
```

#### Purpose
Searches all subfolders for the file.

### 📍 Step 6: Navigate to the File
Use the discovered path with:
```
cd <path>
```

Then confirm with:
```
dir
```

### 📖 Step 7: Read File Contents

#### Command
```
type task_brief.txt
```

#### Purpose
Displays the file contents directly in the terminal.

## 🖥️ Task 3: Gathering System Information on Windows

### 🧠 Overview
Security analysts often need quick information about a system.

Common questions include:
- Who is logged in?
- What machine is this?
- Which Windows version is running?
- What network configuration exists?

### 👤 Step 1: Identify Current User

#### Command
```
whoami
```

#### Purpose
Shows the currently logged-in user account.

### 🖥️ Step 2: Get Computer Name

#### Command
```
hostname
```

#### Purpose
Displays the computer name.

### 🪟 Step 3: View System Information

#### Command
```
systeminfo
```

#### Purpose
Displays detailed operating system information.

**🔑 Important Fields**
- OS Name
- OS Version
- System Type

### 🌐 Step 4: View Network Information

#### Command
```
ipconfig
```

#### Purpose
Displays network configuration details.

**🔍 Important Fields**
- IPv4 Address
- Default Gateway

**✅ Answers**
