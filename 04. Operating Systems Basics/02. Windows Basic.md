# 🪟 Windows Basics 

## 📘 Overview

The **Windows Basics** module introduces you to one of the most widely used operating systems: **Microsoft Windows**.

It builds on your understanding of operating systems by showing how concepts like:

- User management  
- File systems  
- Processes  
- Security  

…are applied in a real-world environment.

> 💡 **Core Idea:**  
> Windows provides a user-friendly interface to interact with your computer while managing everything behind the scenes.

---

## 🎯 Learning Objectives

After completing this module, you will be able to:

- Navigate the Windows interface (Desktop, Taskbar, Start Menu)  
- Use File Explorer to manage files and folders  
- Access and configure system settings  
- Use Task Manager to monitor system performance  
- Understand built-in Windows security tools

## 🖥️ Task 2: Exploring the Windows Workspace

### 🔐 Authentication & User Accounts

Before accessing Windows, users must log in.

#### Account Types:

- **Guest** → Limited access  
- **Standard** → Daily usage  
- **Administrator** → Full system control  

### 🖥️ Windows Desktop Components

#### Main Areas

- **Desktop** → Workspace for files & shortcuts  
- **Taskbar** → Access apps, tools, and notifications  

#### 🔑 Key Features

1. Desktop Icons  
2. Start Menu  
3. Search Bar  
4. Task View  
5. Pinned Apps  
6. Network & Audio Settings  
7. Date & Time  
8. Notifications  

### 📂 Start Menu

- Central hub for apps and settings  
- Access power options (restart, shutdown)  
- Quick navigation  

### 🛠️ Built-in Tools

- **File Explorer** → Manage files  
- **Notepad** → Edit text  
- **Calculator / Paint** → Basic utilities  

### 🖥️ System Information

Use **About your PC** to view:

- Device name  
- RAM  
- OS version  

### 📁 File Management

#### Key Concepts

- Hierarchical folder structure  
- File paths (e.g., `C:\Users\Administrator\Desktop`)  
- Organized storage

## ⚙️ Task 3: Configuring and Securing Windows

### 🔄 Updating Applications

#### Windows Update

- Security patches  
- Bug fixes  
- Performance improvements  

### 📥 Installing Applications

Ways to install:

- Microsoft Store  
- Download `.exe` or `.msi` files  

### ❌ Uninstalling Applications

- Settings → Apps  
- Control Panel  
- Built-in uninstallers  

### ⚙️ System Configuration Tools

#### Windows Settings

- Modern configuration interface  

#### Control Panel

- Legacy system management  

### 📊 Task Manager

Monitor system activity:

- **Processes** → Running apps  
- **Performance** → CPU, RAM usage  
- **Users** → Active accounts  
- **Details** → Advanced process info  
- **Services** → Background services  

### 🛡️ Windows Security

#### Key Features

- **Virus & Threat Protection** → Malware detection  
- **Firewall & Network Protection** → Control traffic  
- **App & Browser Control** → Safe usage  
- **Device Security** → Hardware protection  

### 🔥 Windows Defender Firewall

Network protection profiles:

- **Domain** → Organization networks  
- **Private** → Trusted networks  
- **Public** → Untrusted networks
