## **A Quick Note About the Terminal**

The **terminal** is a text-based interface used to control and interact with a Linux system. Instead of clicking through menus and windows, users type commands directly into the system.

Cyber security professionals rely heavily on the terminal because:

- It is faster than navigating graphical interfaces
- It provides more control over the system
- Many security tools are designed to run exclusively in the terminal

At first, the terminal may feel unfamiliar, but with practice, it becomes one of the most powerful and efficient tools available in Linux.

To begin, open the **Terminal** application from the Desktop.

## **Interacting With the Terminal**

Your supervisor mentioned that a file called `mission_brief.txt` is hidden somewhere on the system. Your first task is to locate it and read its contents.

To accomplish this, you will learn several essential Linux commands.

## **Step 1: Determine Your Current Location**

Use the following command to display the current directory: `pwd`

```bash
Example:
ubuntu@tryhackme:~$ pwd
/home/ubuntu
```
> The pwd command stands for print working directory and displays the folder you are currently operating in.

## **Step 2: View Files and Folders**

To list files and folders in the current directory, use: `ls`

```bash
Example:
ubuntu@tryhackme:~$ ls
Desktop    Downloads  Pictures  Templates  logs
Documents  Music      Public    Videos     projects
```

```
For a detailed listing that includes permissions, ownership, file sizes, and timestamps, use:
ls -l
```

```
To include hidden files and directories, use:
ls -al
```
> Hidden files in Linux begin with a period (.) and are hidden by default.

## **Step 3: Navigate Between Directories**

To move into a directory, use the cd command: `cd Documents`

```bash
Example:
ubuntu@tryhackme:~$ cd Documents/
ubuntu@tryhackme:~/Documents$
```

To move back one directory level: `cd ..`

```bash
Example:
ubuntu@tryhackme:~/Documents$ cd ..
ubuntu@tryhackme:~$
```

## **Step 4: Locate Files Using find**

Linux provides the find utility to search for files and directories.

To search for mission_brief.txt inside the home directory: `find ~ -name mission_brief.txt`

```bash
Example:
ubuntu@tryhackme:~$ find ~ -name mission_brief.txt
/home/ubuntu/Documents/.research/archive/mission_brief.txt
```

The command searches recursively and returns the full path of the matching file.

After locating the file, navigate to its directory using cd.

## **Step 5: Read File Contents**

Use the cat command to display the contents of a file: `cat mission_brief.txt`

```bash
Example:
ubuntu@tryhackme:~/<REDACTED-PATH>$ cat mission_brief.txt
Great job finding your way around the terminal.

Your next assignment is to collect a small system report:
- Who you're logged in as
- The kernel version
- Total disk space
- The name of this Linux distribution

FLAG:MISSION-FOUND
```
> The cat command is commonly used to quickly read text files directly from the terminal.

## **Task 3 : Investigating the System**

Your supervisor now wants you to gather important information about the Linux system you are working on. System administrators and security professionals frequently collect this information to understand the environment, troubleshoot issues, and verify system configurations.

This task introduces several useful Linux commands for gathering system information.

## **Step 1: Identify the Current User**

To display the currently logged-in username, use: `whoami`

```bash
Example:
ubuntu@tryhackme:~$ whoami
ubuntu
```
> This command is useful for confirming which account is currently active.

## **Step 2: Display System Information**

To retrieve information about the operating system and kernel, use: `uname -a`

```bash
Example:
ubuntu@tryhackme:~$ uname -a
Linux tryhackme 6.14.0-1018-aws #17-Ubuntu SMP Mon Sep 2 13:48:07 UTC 2024 x86_64 x86_64 x86_64 GNU/Linux
```

The output includes:
- Operating system name
- Hostname
- Kernel version
- System architecture
- GNU/Linux identifier

If only the operating system name is needed, the following command can be used: `uname`

## **Step 3: Check Disk Usage**

To display disk space information in a human-readable format, use: `df -h`

```bash
Example:
ubuntu@tryhackme:~$ df -h
Filesystem      Size  Used Avail Use% Mounted on
/dev/root        70G   12G    58G  17% /
```

The `-h` flag displays storage sizes using readable units such as MB and GB.

This command helps administrators verify available storage and monitor disk usage.

## **Step 4: Read Operating System Information**

Linux stores many configuration and informational files inside the `/etc` directory.

- Navigate into /etc: `cd /etc`
- List the contents: `ls`
- To display Linux distribution details, read the `os-release` file: `cat os-release`

```bash
Example:
PRETTY_NAME="Ubuntu 24.04.1 LTS"
NAME="Ubuntu"
VERSION_ID="24.04"
VERSION="24.04.1 LTS (Noble Numbat)"
```
This file provides detailed information about the installed Linux distribution.

## **Key Terminology**
- CLI (Command-Line Interface): A text-based interface used to interact with the operating system
- Terminal: The application used to access the Linux command line
- pwd: Displays the current working directory
- ls: Lists files and directories
- cd: Changes the current directory
- find: Searches for files and directories
- cat: Displays the contents of a file
- whoami: Shows the currently logged-in user
- uname: Displays operating system and kernel information
- df -h: Displays disk usage and available storage space
- /etc: Directory containing important system configuration files
